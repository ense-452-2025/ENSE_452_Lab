## 🧠 What Is Git?

- A **version control system**
- Tracks **changes in files**
- Helps you **collaborate** on code
- Commonly used with **GitHub**

---

## 🛠 Installing Git

- Download from [git-scm.com](https://git-scm.com)
- Verify install:

```bash
git --version
```
